## Project Overview

The **Data Science Toolkit** is an interactive web application built using Python and Streamlit to perform common data analysis and financial calculations.

This tool is designed for quick exploratory analysis and basic statistical insights, making it useful for students, analysts, and business users.

---

## Features

### 1. Statistical Calculator

* Mean
* Median
* Standard Deviation
* Variance
* Minimum & Maximum

### 2. Loan EMI Calculator

* Monthly EMI calculation
* Total payment
* Total interest

### 3. CSV Data Analyzer

* Upload CSV files
* Data preview
* Summary statistics
* Missing value analysis

* Python
* Streamlit
* Pandas
* NumPy

